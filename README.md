# Vue3 + TypeScript + Viteテンプレートプロジェクト
[![coverage](https://nashiusagi.github.io/vue3-template/coverage.svg)](https://nashiusagi.github.io/vue3-template/)

## 環境
* Vue3
* TypeScript
* Vite
* ELement-Plus
* Node >= 18

## 実行方法
```
$ yarn dev
```
初回は
```
$ yarn install
```
が必要

## ESLint
以下で設定の一覧を取得できる
```bash
$ yarn eslint --print-config src/App.vue > config.json
```
